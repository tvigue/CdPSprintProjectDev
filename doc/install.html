Pré-requis

Les supports nécessaires pour le bon fonctionnement du site, côté serveur et côté client, sont les suivants :

  • Le serveur HTTP Apache.
  • Le système de gestion de bases de données MySQL.
  • L'outil d'administration web des bases MySQL phpMyAdmin.
  • L'interpréteur de script PHP.
  • Un éditeur de texte.
  • Un navigateur internet.

Comme dans un premier temps ce projet web sera local à votre espace de travail, vous pouvez en fait retrouver les quatre premiers outils dans une plateforme de développement web appelée "Wamp", "Lamp" ou "Mamp"" selon sur quel système d'exploitation vous prévoyez de déposer votre projet. Wamp est pour Windows, Lamp pour Linux et Mamp pour Mac.

Installation

Pour l'importation du site, il vous faut faire les étapes suivantes :

  1. Mettre en place la base de données nommée projet_cdp2016_acv et la table de données ateliers dans votre serveur de base de données. Cette étape se réalise sur l'interface web de phpMyAdmin en important dessus le fichier "database.sql" qui se trouve dans le dossier doc/ du projet.
  2. Vérifier les paramètres de configuration de connexion à votre nouvelle base de données. Pour cela, il faut ouvrir le localhost sur votre navigateur puis atteindre un des documents php à la source de votre projet (dossier src/web/) et voir si cela s'exécute correctement car si ce n'est pas le cas, il suffit d'aller trouver dans le dossier includes/ le fichier config.php et de l'éditer en comprenant à quoi servent les différentes variables de connexion à une BDD.

Utilisation

Normalement, vous n'avez plus qu'à vous balader sur votre site internet qui, soit-dit en passant est en cours de réalisation et donc pas fonctionnel à 100%.
